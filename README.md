# kOS scripts for my playthrough

Collection of kOS scripts i wrote for my ksp playthrough.

If you wish to run any of those scripts, copy this repo to
`.../Kerbal Space Program/Ships/Scripts` and select one of the boot files in your Kos computer.

You will also need [kOS: Kerbal Operating System](https://ksp-kos.github.io/KOS_DOC/) mod.
